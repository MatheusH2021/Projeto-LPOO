# Projeto-LPOO - Agência de Correios
## Desrição: 
### O projeto consiste em simular um sistema de uma agencia de correios, onde possui o modulo de gerente, entregador e funcionario.
* O Gerente é responsavel por gerenciar os funcionarios.
* O Entregador gerencia suas entregas.
* E o Funcionario é responsável por cadastrar e gerenciar encomendas e atribuí-las aos entregadores.
## Como Usar:
### Para iniciar o sistema vc receberá instruções para realizar o cadastro do seu acesso de gerente, e a partir daí será possivel utilizar os outros modulos, uma vez que realize cadastro de novos funcionarios (Entregadores e Funcionarios) você poderá acessar os outros modulos com as credenciais cadastradas. OBS: Só é acessado cada modulo pelo tipo de usuario específico.
